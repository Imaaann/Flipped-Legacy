# Flipped Legacy

## Story Based Card game creator

This software allows the creation of visual novels that have card battle parts. It is divided into 2 parts which are:

- Flipped Legacy Editor:
  This is where you can make games that are split into levels and each level contains a story component and a combat part.
  You will create everything from the characters/enemies and their decks and story graph for each level.
  After creating your game it will be compiled into a .flgx file that can be shared
- Flipped Legacy Launcher:
  This is where you can launch and play the diffrent games made by the editors, all you need is to open a .flgx file or open the launcher and choose the game there

### Notes

- This project is still in beta, the focus of these first version is to make the logic in a terminal interface for the editor
  a UI will be added later
